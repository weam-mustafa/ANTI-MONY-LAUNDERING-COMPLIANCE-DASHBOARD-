Overview

This project showcases an end-to-end Anti-Money Laundering (AML) analytics solution developed using Microsoft Power BI.
The dashboard is designed to support KYC, customer risk assessment, transaction monitoring, and AML alert analysis in a regulated banking environment.
The project demonstrates how Power BI can be used to translate financial and compliance data into actionable insights for compliance teams, investigators, and management. Project Objectives
Analyze customer risk profiles (low, medium, high)
Support KYC / CDD / EDD decision-making
Monitor transaction behavior and identify unusual patterns
Track and prioritize AML alerts
Enable drill-through customer investigations
Provide executive-level AML reporting
